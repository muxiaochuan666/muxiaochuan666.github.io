## 📖 项目简介

**HotVideo** 是一款在手机上独立开发完成的安卓热点视频 APP，集成 **仿抖音视频流**、**AI 聊天室**、**全网热榜** 三大核心功能。从 V1 到 V8，经过 8 个版本迭代打磨。

本站点 (`muxiaochuan666.github.io`) 是 HotVideo 的官方下载页，使用 GitHub Pages 纯静态部署，**完全免费、无需备案**。

## ✨ 核心功能

| 模块 | 说明 |
|------|------|
| 🎬 **仿抖音视频流** | 上下滑动无限刷新，沉浸式全屏播放，点赞、评论、分享一键直达 |
| 💬 **AI 聊天室** | 内置智能助手，陪聊、推荐你感兴趣的热点内容 |
| 🔥 **全网热榜** | 实时聚合各大平台热搜，5 分钟刷新一次，热点不再错过 |
| 📱 **纯手机开发** | 全程在 Android 手机上使用 AIDE 编码、调试、签名打包 |
